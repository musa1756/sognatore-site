# Edward Sexton Homepage Behaviors

- Header is sticky at the top. Source uses `.section-header { position: sticky; }`; clone keeps the same behavior.
- Desktop nav items fade on hover. Clone uses opacity transition.
- Mobile nav opens from a hamburger. Source uses JavaScript; clone uses a CSS checkbox drawer to keep the behavior without backend/state dependencies.
- Hero has a static overlay image and two CTA buttons. Buttons invert from cream fill to transparent on hover.
- Gallery cards zoom image slightly on hover and retain a dark lower gradient for legibility.
- Mobile gallery rows behave as carousels. Source uses Slick classes; clone uses native horizontal scroll snap.
- Footer newsletter is visual only. Real Shopify form submission is out of scope for this static clone.
