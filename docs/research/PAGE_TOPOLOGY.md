# Edward Sexton Homepage Topology

Source URL: https://www.edwardsexton.co.uk/

1. Announcement bar
   - Dark green strip, 34px tall, currency disclosure on the right at desktop and centered on mobile.
   - Interaction: click target only in source; cloned as static disclosure button.

2. Sticky header
   - Cream background, centered SVG logo, desktop navigation on the left, account/search/cart icons on the right.
   - Mobile: hamburger opens left drawer; logo width reduces from 250px to 180px.
   - Interaction: sticky scroll, hover opacity on nav, CSS mobile drawer.

3. Hero image overlay
   - Desktop image: Edward-sexton-collection.jpg. Mobile image: Summer-tailoring.
   - Fixed height: 680px desktop, 460px mobile.
   - Centered serif title "Spring Summer", paragraph, two cream buttons.

4. Collection gallery rows
   - Two-column desktop grid, 4:5 cards with dark gradient overlay.
   - Mobile horizontal carousel behavior per source class `gallery--mobile-carousel`; cloned with native horizontal scroll snap.
   - Cards: Bespoke, Ready To Wear, Jackets, Suits, Shirts, Black Tie, Casual Wear, Discover Edward Sexton.

5. Women's bespoke banner
   - Single wide image row, 550px desktop, 270px mobile source height.
   - Overlay text: Couture Craft and Women's Bespoke CTA.

6. Footer
   - Dark green background, four columns: brand copy, newsletter/social links, information, policies.
   - Bottom copyright line.
